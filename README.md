# Get Optifine Download Link (GOFDLL)

> A scraper to get a file download (request) link from optifine.net

## Usage

```ts
import gofdll from 'gofdll';

gofdll.getDownloadLink('OptiFine_1.19.2_HD_U_H9.jar');
```
